# Doki Doki Fantasista SQUAD
DDLC but with Fantasista SQUAD<br>
After starting a new game, wait 1.5 seconds to be redirected to the poem game. If you'd like to skip the <a href="https://asiandra.github.io/dokidokifantasistasquad-menu/">main menu</a>, <a href="https://asiandra.github.io/dokidokifantasistasquad-menu/Stuff/poemgame.html">click here</a>.<br>
If you'd like to <s>cheat</s> see what words are associated with who, the list can be found <a href="https://docs.google.com/document/d/1eWWXi6MVdbkCTmaBG0MU7N07LInZP-A-eMsVTGOGJpQ">here</a>. You could also use the hints button; it was mainly for dev use but I'm too lazy to remove it.
